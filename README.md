# card-game

Минимальный headless-каркас карточного движка на TypeScript.

## Быстрый обзор
- Фиксированные фазы: draw → main → combat → end.
- Детерминированный RNG с сидом.
- Событийная шина с приоритетами.
- Разрешение эффектов в упорядоченном виде.
- Переход активного игрока по фиксированному порядку.
