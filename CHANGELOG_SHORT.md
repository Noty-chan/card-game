# Краткий журнал изменений

- Добавлены правила завершения игры, новые поля состояния, события и тесты.
- Добавлены сериализация/десериализация состояния, детерминированный формат и тесты.
- Добавлена трассировка событий движка с буфером, API и тестами.
- Добавлены базовые типы и пайплайн команд (валидация/применение) с событиями и тестами.
- Добавлен базовый каркас headless-ядра (фазы, события, эффекты, RNG).
- Добавлены русские инструкции для агента и обновлён контекст проекта.
- Добавлена возможность подключать плагины через конфигурацию движка.
- Добавлен файл с коротким планом завершения проекта.
- Зафиксирован API модулей правил и их подключение в конфигурации движка.
- Добавлен порядок игроков и автоматическая смена активного игрока между ходами.
- Добавлена документация публичного API движка.
- Добавлены проверка детерминизма и скрипт измерения размера headless-ядра.
- Добавлены пример данных карт JSON5 и документация по правилам и инициализации.
- Добавлена нормализация конфигурации движка и обновлена документация по API.
- Проведён аудит размера headless-ядра и добавлен отчёт.
- Добавлен лимит глубины цепочки событий и тест защиты от бесконечной рекурсии.
- Добавлен детерминированный планировщик отложенных действий по фазам/ходам и его тесты.
